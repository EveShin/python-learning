# 🐍 파이썬 학습 저장소

파이썬 프로그래밍 언어를 체계적으로 학습하기 위한 개인 저장소입니다.

## 📚 학습 목표

- 파이썬 기초 문법 완전 정복
- 객체지향 프로그래밍 개념 이해
- 실무에 활용할 수 있는 라이브러리 활용법 습득
- 실제 프로젝트를 통한 실전 경험 쌓기

## 📖 학습 계획

### 1단계: 기초 문법 (1-2주)
- [ ] 변수와 데이터 타입
- [ ] 연산자
- [ ] 조건문 (if, elif, else)
- [ ] 반복문 (for, while)
- [ ] 입출력 함수

### 2단계: 자료구조 (1-2주)
- [ ] 리스트 (List)
- [ ] 튜플 (Tuple)
- [ ] 딕셔너리 (Dictionary)
- [ ] 집합 (Set)
- [ ] 문자열 처리

### 3단계: 함수와 모듈 (1-2주)
- [ ] 함수 정의와 호출
- [ ] 매개변수와 인자
- [ ] 람다 함수
- [ ] 모듈 import
- [ ] 패키지 만들기

### 4단계: 객체지향 프로그래밍 (2-3주)
- [ ] 클래스와 객체
- [ ] 상속
- [ ] 캡슐화
- [ ] 다형성
- [ ] 특수 메소드

### 5단계: 고급 개념 (2-3주)
- [ ] 파일 입출력
- [ ] 예외 처리
- [ ] 데코레이터
- [ ] 제너레이터
- [ ] 정규표현식

### 6단계: 라이브러리 활용 (2-3주)
- [ ] NumPy (수치 계산)
- [ ] Pandas (데이터 분석)
- [ ] Matplotlib (시각화)
- [ ] Requests (HTTP 요청)
- [ ] BeautifulSoup (웹 스크래핑)
